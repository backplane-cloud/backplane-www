Backplane is developed using a microservices design architecture.

### Problem Statement

- Multi Cloud governance is complicated
- Multi Cloud integration is complex
- Cloud Cost visibility across multiple clouds is poor
- Cloud Permission Posture is difficult to assess in multiple clouds
- A high degree of subject matter expertise is required in multiple clouds
- Avoid vendor lock-in
- maximise value in cloud

### Requirements

- Multi-cloud cost transparency
- Multi-Cloud governance

### Architect (Design Decisions)

Microservices, Event Driven. Restful.
HTMX. JSON.
